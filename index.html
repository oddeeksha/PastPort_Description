PastPort

Messages That Wait for the Right Moment

Why I Chose This Project

I wanted to create something meaningful yet funâ€”a way to connect with the future and preserve memories. This project allows users to send messages or images to their future selves, giving them a unique surprise waiting at just the right moment.

Features

  • Upload messages and images to a digital time capsule.
  • Set a specific date and time for the capsule to open.
  • Securely store memories in the cloud.
  • Receive reminders when the time capsule is close to opening.
  • Prevent time manipulation with server-side validation.

How It Works

  1. Users upload their message or image to the cloud.
  2. Set the desired unlock date and time.
  3. The data remains locked until the specified time.
  4. Receive the content back when the time comes.

Technology Used

  • HTML and CSS: For the structure and styling of the web pages.
  • Firebase: For backend services, including Firestore for the database and Firebase Storage for file uploads.
  • JavaScript: To add functionality and handle interactions with the cloud backend.
  • Google Cloud Platform (GCP): For hosting and other cloud services.
